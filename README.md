# Microsoft-Power-BI-Dashboard
Interactive Sales Dashboard | Power BI Project 
This interactive Sales Dashboard built in Power BI dynamically visualizes key sales insights, enabling better decision-making through interactive filters and drill-down capabilities.

Key Features:

Real-time Sales Insights – Track total sales, cost, and quantity at a glance
Brand Performance Analysis – Compare sales across different brands
Geographical Sales Distribution – Interactive map visualization
Product-Wise Insights – Dynamic pie charts for product quantity analysis
Fully Interactive Dashboard – Users can filter and explore data effortlessly

Tools Used: Power BI, DAX, Data Modeling
Built for: Sales Analysis, Business Intelligence, and Data-Driven Decision Making
